# Player de Música

### Frontend

#### Para rodar
- Entrar na pasta `front` e executar

```
npm install

npm start
```

### Backend
- Versão do Python `3.7`
- Com Gerenciador de dependencias `PIPENV`

#### Para instalar
- Entrar na pasta `api`

```
pipenv install
```

#### Para rodar 

```
pipenv shell
python main.py
```
